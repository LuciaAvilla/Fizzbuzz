## Sobre o Projeto

Este projeto é uma variação do clássico desafio **FizzBuzz**, muito utilizado para praticar lógica de programação.

O programa solicita que o usuário digite um número e, em seguida, percorre todos os números de `1` até o valor informado, aplicando regras específicas para exibir diferentes resultados.

---

## Funcionalidades

O código analisa cada número e retorna:

* `FizzBuzz` → quando o número é divisível por **3 e 7**
* `Fizz` → quando o número é divisível por **3**
* `Buzz` → quando o número é divisível por **7**
* `Almost Fizz` → quando o número contém o dígito **3**
* O próprio número → caso nenhuma regra seja atendida

---

## Lógica Utilizada

O programa utiliza:

* Funções em Python
* Estruturas condicionais (`if`)
* Operador módulo (`%`) para verificar divisibilidade
* Conversão de número para texto usando `str()`
* Estrutura de repetição `for`
* Entrada de dados com `input()`
